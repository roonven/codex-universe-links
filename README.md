# Codex Universe Links

A futuristic personal link hub built with Astro for RooWeb.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Edit Content

- `content/profile.json`
- `content/links.json`

## Deploy

The repository includes `.github/workflows/deploy-pages.yml` for GitHub Pages. Set the GitHub repo to use GitHub Actions as the Pages source.
